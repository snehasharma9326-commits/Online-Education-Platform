Variables & Operators

Variables are used to store data values.

int a = 10;
float b = 5.5;
char c = 'A';

If Else Statement

if(a > b)
{
    printf("A is greater");
}
else
{
    printf("B is greater");
}

Loops

for(int i=1;i<=5;i++)
{
    printf("%d", i);
}

Pattern Printing

for(int i=1;i<=5;i++)
{
    for(int j=1;j<=i;j++)
    {
        printf("*");
    }
    printf("\n");
}

Functions

int add(int a, int b)
{
    return a+b;
}

Array

int arr[5] = {1,2,3,4,5};

2D Array

int a[2][2] = {{1,2},{3,4}};
