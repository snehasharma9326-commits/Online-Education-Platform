CSS Introduction

CSS stands for Cascading Style Sheets.
It is used to style and design HTML pages.

CSS Syntax

h1 {
  color: red;
  font-size: 30px;
}

CSS Selectors

#idSelector { }
.classSelector { }
p { }

Colors & Background

body {
  background-color: lightblue;
}

Box Model

div {
  margin: 10px;
  padding: 20px;
  border: 2px solid black;
}

Layout

.box {
  display: inline-block;
  position: relative;
}

Flexbox

.container {
  display: flex;
  justify-content: center;
}

Responsive Design

@media (max-width:600px) {
  body {
    background: lightgray;
  }
}
